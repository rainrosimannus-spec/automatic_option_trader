Merit — quarterly loan summary

One row per active loan per quarter: opening principal, disbursements, repayments, restructures, closing principal, interest accrued, closing accrued interest. Use these to post journals for the period.

Merit reconciliation

After the bookkeeper posts the quarter, head to /borrower/merit-reconcile to diff Bruno's closing outstandings against Merit Aktiva's closing balances. Import a Merit CSV export there; on Rasmus's production clone it's pulled automatically via the Merit API.
